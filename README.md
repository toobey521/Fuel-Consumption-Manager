# ⛽ 油耗管家

车辆油耗记录与管理工具，帮助分析油耗趋势和费用。

## 功能

- 📝 **加油记录**：记录每次加油的金额、油量、里程
- 📊 **油耗分析**：计算百公里油耗、费用统计
- 📈 **趋势图表**：油耗变化趋势可视化
- 💰 **费用统计**：月度/年度油费汇总

## 快速开始

1. 添加加油记录
2. 查看油耗统计
3. 分析趋势

## 技术栈

- 前端: 原生 HTML5 / CSS3 / JavaScript
- 部署: GitHub Pages
- LocalStorage 数据存储
